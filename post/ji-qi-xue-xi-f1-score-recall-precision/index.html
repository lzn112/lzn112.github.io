机器学习 F1-Score, recall, precision

2024-09-08

Acc=Npre/Ntotal

true positive (TP)​ pre = 1 true =1
false positive (FP) pre = 1 true = -1
false negative (FN) pre = -1 true = 1
true negative (TN) pre = -1 true =-1

预测正确数： Npre = TP+TN
总的预测数： Ntotal = TP + TN + FP + FN

  • Acc= TP+TN/TP+TN+FP+FN = TP+TN/P+N

​- Recall= TP/TP+FN = TP/P
召回率就真样本被成功预测（召回）的比例
表现对正样本的识别能力

  • Precision= TP/TP+FP
    Precision就是预测的正样本中真实的比例
    表现对负样本的识别能力

F1= 2TP/2TP+FN+FP = 2⋅Precision⋅Recall/Precision+Recall
