Multi-View Spatial-Temporal Graph Convolutional Networks With Domain Generalization for Sleep Stage Classification

2024-09-24

contribution

  • We construct different brain views based on the functional connectivity and physical distance proximity of the brain. The complementarity of different views provides rich spatial topology information for classification tasks. （采用物理连接和功能连接俩个视图进行图卷积，最后进行融合）
  • We design a spatial-temporal graph convolution with attention mechanism, which consists of spatial-temporal graph convolution for spatial-temporal features and attention mechanism for capturing the most relevant spatial-temporal information for sleep stage classification. （采用时间空间俩种注意力机制）
  • We integrate domain generalization and spatial-temporal graph convolutional networks into a unified framework to extract subject-invariant sleep features. （采用域对抗泛化的方法提高跨被试的性能）
  • We conduct experiments on two public sleep datasets, namely ISRUC-S3 and MASS-SS3. Experimental results demonstrate that the proposed model achieves the stateof-the-art performance. （代码跑出来的准确率确实不错）
  • We explore the interpretability of the key modules of the model. In particular, we present the functional connectivity obtained through adaptive graph learning. The results indicate that functional connectivity during light sleep is more complex than that during deep sleep.（实验的结果也体现了一定的可解释性）

图学习损失函数

Lgraph_learning=∑m,n=1N∥xm−xn∥22 AFCmn+λ∥AFC∥F2L_{\text{graph\_learning}} = \sum_{m,n=1}^{N} \|x_m - x_n\|_2^2 \, A_{FC}^{mn} + \lambda \|A_{FC}\|_F^2 Lgraph_learning​=m,n=1∑N​∥xm​−xn​∥22​AFCmn​+λ∥AFC​∥F2​

公式解析：

  • ( L_{\text{graph_learning}} ): 图学习的损失函数。
  • ( \sum_{m,n=1}^{N} |x_m - x_n|2^2 , A{FC}^{mn} ):
    • 衡量节点 ( m ) 和节点 ( n ) 特征差异的加权和，其中：
    • ( |x_m - x_n|_2^2 ): 节点 ( m ) 和节点 ( n ) 的特征差异（欧氏距离的平方）。
    • ( A_{FC}^{mn} ): 节点 ( m ) 和节点 ( n ) 之间的相似度权重（邻接矩阵中的元素）。
  • ( \lambda |A_{FC}|_F^2 ):
    • 邻接矩阵 ( A_{FC} ) 的 Frobenius 范数的平方，用于正则化。
    • ( \lambda ): 正则化系数，控制邻接矩阵的稀疏性，防止过拟合。

公式解释：

  1. 第一项：度量节点特征差异与邻接矩阵权重的关系，反映了相似节点应该有更高的连接权重。

  2. 第二项：邻接矩阵的正则化项，控制图结构的复杂度，避免邻接矩阵中的过度连接。
